Technical Pipeline
INPUT → Food Image (JPEG/PNG, max 16MB)
→ YOLO Bounding box detection [planned]
→ EfficientNetV2-S + Swin-T + ViT-B/16 ensemble
→ SOFTMAX → [fresh, semi_fresh, rotten, cooked, packaged]
→ DECISION ENGINE → confidence threshold check
→ MATCHING ENGINE → Haversine distance + capacity score
→ OUTPUT → Destination + nearest facility + route steps
