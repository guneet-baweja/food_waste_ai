# AI Powered Food Waste Management Platform

## Overview

This project is an immersive, cinematic, scroll-driven 3D website designed for the "AI Powered Food Waste Management Platform." It utilizes advanced technologies to create a visually stunning experience that combines storytelling with real-time 3D environments.

## Tech Stack

- **Three.js**: Core 3D engine for rendering graphics.
- **GSAP + ScrollTrigger**: For smooth scroll animations and timeline control.
- **WebGL Shaders**: For advanced visual effects.
- **React**: Framework for building the user interface.
- **Framer Motion** (optional): For additional animations.
- **GLSL Shaders**: For particles, fog, and distortions.

## Features

- **Cinematic Journey**: The entire website behaves like a cinematic experience, with smooth scrolling and frame-by-frame control.
- **Dynamic Background System**: Includes floating particles, volumetric fog, animated grids, nebula clouds, and a 3D geometry field.
- **Hero Section**: Features a floating 3D Earth with glowing network lines and food objects orbiting it.
- **AI Visualization Scene**: Displays a glowing neural brain with particle animations representing food inputs and outputs.
- **Floating Dashboard**: Glassmorphism panels with interactive 3D cards and depth tracking.
- **Scroll-Triggered Animations**: Includes camera fly-throughs, object assembly, and parallax layers.
- **Mouse Interactions**: Magnetic buttons, 3D tilt cards, and particle repulsion effects.
- **Product/Feature Scenes**: 3D rotating UI panels and exploded view animations.
- **Performance Optimization**: Uses instancing for particles, lazy loading, and texture compression to maintain 60 FPS.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ai-food-waste-3d
   ```

3. Install the dependencies:
   ```
   npm install
   ```

### Running the Project

To start the development server, run:
```
npm run dev
```

Open your browser and navigate to `http://localhost:3000` to view the application.

### Building for Production

To build the project for production, run:
```
npm run build
```

The production files will be generated in the `dist` directory.
